# CloudCart Enterprise Platform

> Production-grade cloud-native e-commerce platform built with Python, FastAPI, React, Kubernetes, Terraform, Chef, Prometheus, Grafana, and Google Cloud Platform.

---

## 📖 Overview

CloudCart Enterprise Platform is a full-stack cloud-native e-commerce application designed to simulate a real-world enterprise production environment.

The objective of this project is not only to build an online shopping application, but also to demonstrate modern software engineering, DevOps, Site Reliability Engineering (SRE), cloud infrastructure, monitoring, observability, CI/CD, and production operations.
